# Update the Properties of an Object

This generic function allows for updating the properties or state of an
object by applying changes from another object. It ensures that the
input object is modified according to the specifications of the provided
`object`.

## Usage

``` r
ggalign_update(x, object, ...)
```

## Arguments

- x:

  The object whose properties are being updated.

- object:

  The object from which the properties will be applied to `x`.

- ...:

  Arguments passed to methods.
