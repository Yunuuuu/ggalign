# Convert the shape of a matrix for fortify method

Convert the shape of a matrix for fortify method

## Usage

``` r
# S3 method for class 'matrix'
tune(data, shape)
```

## Arguments

- data:

  A matrix.

- shape:

  A string of `"upset"` or `"oncoplot"`.
